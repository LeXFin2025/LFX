# LeXFin

LeXFin is an AI-powered legal and financial advisory platform that provides intelligent, comprehensive guidance across complex regulatory landscapes through advanced technology and user-centric design.

## Features

- **AI-Powered Legal and Financial Analysis**: Comprehensive document analysis with jurisdiction-specific insights
- **LeXAssist Chat**: Interactive AI assistant for legal and financial advice
- **Document Management**: Upload and analyze tax, legal, and forensic documents
- **Time Machineâ„¢**: Predict future legal and financial risks with AI-powered simulation
- **Jurisdiction-Specific**: Special focus on Indian laws and regulations
- **User-Friendly Interface**: Modern, responsive design with intuitive navigation

## Technology Stack

- **Frontend**: React.js with TypeScript, TanStack Query, Shadcn/UI components
- **Backend**: Express.js with TypeScript
- **AI Integration**: Google Gemini API
- **Database**: PostgreSQL
- **Authentication**: Passport.js with local strategy

## Getting Started

### Prerequisites

- Node.js 16+ installed
- PostgreSQL database

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/lexfin.git
cd lexfin
```

2. Install dependencies
```bash
npm install
```

3. Set up environment variables
Create a `.env` file in the root directory with the following variables:
```
DATABASE_URL=postgresql://user:password@localhost:5432/lexfin
GEMINI_API_KEY=your_gemini_api_key
SESSION_SECRET=your_session_secret
```

4. Start the development server
```bash
npm run dev
```
